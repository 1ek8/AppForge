## 1. Backend
 (Express)

The backend will handle storing, retrieving, adding, and deleting to-do items. We'll use a simple in-memory array for storage for this example.

**Navigate to the `server` directory:**

```bash

mkdir todo-app
cd todo-app
mkdir server
cd server
```

**Initialize Node.js project and install Express:**

```bash
npm init -y
npm install express cors
```

**`server/index.js`:**

```javascript
const express = require('express');

const cors = require('cors'); // Import cors

const app = express();
const PORT = process.env.PORT || 5000;

// Middleware
app.use(cors()); // Enable CORS for all routes
app.use(express.json()); // Body parser for JSON requests

memory "database" for todos
